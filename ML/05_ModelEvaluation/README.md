## 일반
문제 정의 단계에서 설정한 평가지표로 모델의 성능을 살펴볼 차례이다.
단지 모델점수만 보는것이 아닌 특성중요도나 오차분석도 시행해야 모델 해석에 용이하다.
이를 통해 인사이트를 얻고 전처리를 거듭하여 수정해나간다.
하이퍼파라미터 조정은 모델보다는 전처리에 쓰인 변수 수치를 조정하는 것을 추천한다.

이 단계에서 눈 여겨볼 점은 아래와 갘다.
* 모델의 평가 점수와 그 오차
* (학습데이터가 큰 경우)모델 학습, 추론 시간
* 검증데이터의 특성중요도
* 오차분포도

## 고급
모델을 해석하는 데에는 주로 SHAP가 주로 쓰인다.
이는 국소적인 해석과 전역적인 해석에 신뢰도가 높기 때문이다.
그렇지만 특성과 또 다른 특성이 강한 연관이 있는 다중공선성을 주의해야한다.
다중공선성은 특성중요도와 SHAP에 왜곡된 결과를 보이며 잘못된 가정을 야기하기 때문이다.
또한, 실제 데이터는 시간이 지남에 따라 공변량도 바뀌기에 자연스레 모델성능은 하락한다.
때문에 모델을 배포한다면 주기적으로 재학습하고 모니터링을 해야한다.
