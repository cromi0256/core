## 일반
머신러닝 모델을 만들기 전에 몇가지 확인해야 할 사항이 몇가지 있다.
아래 사항은 코드 실행에 필요한 기본요소다
- 데이터로 어떤 비즈니스 목적(분류, 회귀, 군집화 등)을 이룰 것인가
- 사용할 수 있는 알고리즘은 어떤 것이 있는가
- 모델 평가에 어떤 성능 지표를 쓸 것인가
- 어느정도의 컴퓨팅 성능을 요구하는가

## 고급
만약, 비공개 내부테스트 용도가 아니라면 추가적으로 점검할 사항이 있다.
- 데이터 또는 알고리즘의 라이센스와 접근권한을 확인한다
- 데이터에 민감정보가 포함되어 있는지 확인한다
- 모델 학습주기, 배포주기를 설정한다
- 데이터, 모델 거버넌스를 준수한다

이 과정을 거친후 학습에 쓰일 데이터와 모델 구축에 필요한 라이브러리를 불러온다.
