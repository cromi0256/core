## 일반
전처리 방식까지 완료했다면 데이터를 분할하고 여러 모델을 학습시킬 차례다.
주의할 점은 전처리는 학습데이터를 기준으로 적용해야 한다는 점이다.(과대적합 문제)
처음부터 고도화된 모델을 만들기 보다는 간단하고 다양한 모델을 실험하는 편이 좋다.
경험론적으로 트리 앙상블계열이 주로 쓰인다.

이유는 아래와 같다.
1. 전처리가 간단하다.
  * 결측치를 모델 내부적으로 허용한다
  * 연속형 변수의 경우 표준화하지 않아도 된다
  * 범주형 변수는 OrdinalEncoding으로 강한 차원축소를 할 수 있다
2. 데이터 분포에 가정을 하지않아 비선형 데이터에도 적용할 수 있다
3. 특성이 서로 연관이 있는 경우에도 어느정도 학습할 수 있다
4. 여러 하위 모델을 앙상블하기에 일반화/강건성이 높다

## 고급
모델을 여러개 앙상블 할 수도 있지만 데이터셋을 앙상블 할 수도 있다.
5-Fold 교차검증을 예로 든다면 5번의 교차검증으로 모든 데이터를 학습하면서도 일반화 성능이 향상된다.
이 방법은 Kaggle이나 데이콘에서도 유용한 방법이며 작성자 또한 이 방법으로 입상한 경험이 있다.
머신러닝에서 모델 학습이 중요해보이지만 사실은 좋은 데이터가 좋은 결과를 만드므로, 전처리에 더 신경을 써야한다.
