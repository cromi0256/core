## 일반
탐색적 데이터 분석(EDA)의 목적은 데이터로부터 인사이트를 얻는 것이다.
아래는 정형데이터를 가정한 EDA 절차다.
1. 불러온 데이터의 샘플 수와 특성 수를 확인한다
2. 각 특성들의 통계량을 확인한다
3. 결측치와 이상치가 있는지 확인한다
4. 특성과 타깃의 분포를 확인한다
  - 명목형 변수(그리고 분류 문제)는 막대그래프를 그린다
  - 연속형 볌수(그리고 회귀 문제)는 히스토그램을 그린다
5. 특성과 타깃간의 관계를 확인한다
  - 명목형 특성/이진분류 문제 : 누적막대그래프
  - 연속형 특성/이진분류 문제 : 2개의 히스토그램
  - 명목형 특성/회귀 문제 : 박스플롯 또는 바이올린 플롯
  - 연속형 특성/회귀 문제 : 산점도

## 고급
때론 특수한 경우의 시각화가 필요한 상황도 존재한다.
